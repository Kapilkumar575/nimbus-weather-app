# Nimbus Weather App 🌦️

A modern full-stack weather application built with:

## 🚀 Tech Stack
- React + Vite
- Tailwind CSS
- Node.js + Express
- OpenWeatherMap API

## ✨ Features
- Search weather by city
- Auto location detection
- 5-day forecast
- Weather icons & UI
- Search history
- Backend API proxy (secure API key)
- Caching + rate limiting

cd backend
npm install
node server.js

## 🔑 Environment Variables

Create `.env` in backend:
